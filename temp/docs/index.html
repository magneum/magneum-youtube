VideoMeta()

Parameters:
VideoMeta() needs one paramter
VideoMeta("song-name / youtube-url");
Usage:
VideoMeta("ganja baby", (callback) => console.log(callback));
? callback: callback logs that will be available when the execution is completed

VideoSave()

Parameters:
VideoSave() needs three paramter
VideoSave("song-name / youtube-url", datalink, savepath); Usage:
VideoSave("ganja baby", datalink, path, (callback) => console.log(callback));
? datalink: the link of the video got from VideoMeta()
? path: absolute path where you wish to save the file
? callback: callback logs that will be available when the execution is completed
